## Overview
This task automates the Facebook signup page to practice locating web elements using different Selenium locator strategies and interacting with form fields.
Target URL: https://www.facebook.com/r.php?entry_point=login

## Technologies Used

- Java
- Selenium WebDriver
- TestNG
- Page Object Model (POM)
- Explicit Wait (WebDriverWait)

## Locator Strategies Used

- XPath (with advanced axes such as preceding-sibling and ancestor)
- CSS Selector
- Attribute-based selection
- Text-based XPath
- Dynamic XPath construction

## What Is Implemented

Locate and fill:
- First Name
- Surname
- Date of Birth (Day, Month, Year dropdowns)
- Gender
- Mobile Number
- Password
- Click Submit button
- Navigate to login page

## Purpose

To practice advanced XPath techniques and build a clean, maintainable test structure using the Page Object Model.
